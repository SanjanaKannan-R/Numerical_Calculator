Simpson's 1/3 Rule — Numerical Integration

Approximate the definite integral of f(x) over [a, b] using parabolic arc approximations. Requires an even number of sub-intervals. Accuracy: O(h⁴).

Input

Formula: ∫ f(x) dx ≈ (h/3) [ f(x₀) + 4f(x₁) + 2f(x₂) + 4f(x₃) + ··· + 4f(x_{n-1}) + f(xₙ) ]
where h = (b − a) / n and n must be even.
Available: sin, cos, tan, exp, log, sqrt, abs, pi, e
Python/math syntax: sin(x), x**2, exp(x), log(x), sqrt(x)
